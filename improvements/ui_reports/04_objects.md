# UI/UX 개선 보고서: 오브젝트 (Objects)

오브젝트 페이지는 정책의 구성 요소인 IP, 서비스, 그룹 정보를 관리하는 백과사전과 같은 곳입니다. 방대한 오브젝트 사이에서 원하는 정보를 빠르게 찾고 관계를 파악하기 위한 개선 방안을 제안합니다.

---

## 1. 탭 네비게이션 (Tab Navigation)

### 1.1 탭 스타일 및 활성화 상태
- **현황**: 현재 탭 구분이 다소 평면적이며, 선택된 탭의 시각적 강조가 약합니다.
- **개선**: 
  - 선택된 탭 하단에 2px의 `ds-tertiary` 인디케이터를 추가하십시오.
  - 탭 배경에 미세한 그래디언트나 `bg-ds-surface-container-low`를 적용하여 콘텐츠 영역과의 계층 구조를 명확히 하십시오.

---

## 2. 오브젝트 상세 조회 및 관계 (Object Relations)

### 2.1 상세 모달 (ObjectDetailModal) 고도화
- **현황**: 오브젝트 클릭 시 나타나는 모달이 단순한 텍스트 나열에 그칩니다.
- **개선**: 
  - **Reference(참조)** 기능을 강화하여 해당 오브젝트가 사용된 **방화벽 정책 리스트**를 모달 내에서 즉시 확인할 수 있게 하십시오.
  - 모달 내의 데이터 역시 `font-mono`와 `ds-on-surface-variant` 컬러를 사용하여 시스템의 일관성을 유지하십시오.
  - 모달 배경에 `glass-panel` 효과를 적용하여 깊이감을 부여하십시오.

---

## 3. 그룹 멤버 시각화 (Group Members)

### 3.1 멤버 칩(Chip) 디자인
- **현황**: 그룹 내 멤버들이 `ds-secondary-container` 배경의 칩으로 표시되나, 멤버가 많을 경우 레이아웃이 깨질 수 있습니다.
- **개선**: 
  - 멤버가 5개 이상일 경우 "+N" 형태로 축약하고, 클릭 시 전체 리스트를 보여주는 **Expandable Chip** 구조를 도입하십시오.
  - 각 칩에 마우스 오버 시 해당 멤버 오브젝트의 간략한 정보(IP 등)를 툴팁으로 제공하십시오.

---

## 4. 데이터 필터링 및 내보내기 (Search & Export)

### 4.1 퀵 서치바 (Quick Search)
- **현황**: 테이블 상단의 검색바가 다소 작고 존재감이 부족합니다.
- **개선**: 
  - 검색바 폭을 넓히고 포커스 시 `ds-tertiary` 색상의 글로우 효과를 적용하십시오.
  - 검색 결과가 없을 때의 **Empty State**를 "Precision Sentinel"의 톤앤매너(아이콘 + 안내 문구)에 맞춰 디자인하십시오.

### 4.2 엑셀 내보내기 (Excel Export)
- **현황**: 내보내기 버튼이 일반 버튼과 차별화되지 않습니다.
- **개선**: `ds-secondary-container` 배경을 사용하여 "보조 액션"임을 명시하되, 아이콘과 텍스트의 정렬을 정교하게 다듬어 전문적인 도구 느낌을 주십시오.
